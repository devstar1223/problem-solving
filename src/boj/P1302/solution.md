# 🔍 문제 해석

- 최대 1000인 책의 개수 N이 들어온다.
- 둘째줄 부터 N개의 책 제목이 입력으로 들어온다.
- 책 제목 법칙
  - 길이는 최대 50이다.
  - 알파벳 소문자로만 이루어져 있다.
- 가장 많이 팔린 책 이름을 출력하라.
  - 여러 개일 경우 사전순으로 가장 앞서는 제목을 출력하라.

***

# 🛠️ 풀이

- Map을 사용하는 방법이 꽤 깔끔해 보인다.
  - N이 1000일때, 1000개만큼 입력받고, 최대 1000개의 키를 꺼내서 최대값을 갱신하는것은 시간 범위내에 충분히 가능하다.
  - 최초의 키가 없을 경우 값을 0으로 세팅해 준 후 1을 더해야 하고, 최초의 키가 있다면 바로 1을 더해준다.
  - 이후 `Map` 의 키 목록을 `Set` 으로 받아온다.
  - 최대 팔린 책 개수를 저장하는 `maxCount` 를 선언한다.
  - `Set`을 순회하며 `Map`에서 값을 가져오고 그 값으로 `maxCount`를 갱신한다.
  - 같은 책 개수를 고려하여, `maxCount` 조건에 따라 정답 `List`를 비우거나 추가한다.
  - 마지막 정답을 출력하기 이전, 오름차순 하고 맨 첫번쨰 값을 정답으로 출력한다.

***

# ⏱️ 시간 복잡도

시간 복잡도 : $O(N*logN)$
- $N$번 순회 + $N*logN$ 시간복잡도로 정렬 -> $N+NlogN$ -> $O(N*logN)$
